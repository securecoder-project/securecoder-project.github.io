SecureCoder JetBrains Plugin

Bringing security-aware code generation natively to your IDE

Overview

The SecureCoder JetBrains plugin acts as your secure copilot. Traditional LLM-based coding assistants can inadvertently generate vulnerable code, ranging from buffer overflows to SQL injections. The SecureCoder plugin integrates directly into the developer workflow, providing real-time security context analysis.

Under The Hood

Our plugin is powered by advanced methodologies outlined in our research proposal:

  • Graph Neural Networks (GNNs): The plugin assists in parsing the context logic around your cursor to maintain structural integrity across generated tokens, preventing security oversights when writing complex software logic.
  • Zero-Shot Constraint Decoding: Enforces security rules directly at generation time without necessarily relying exclusively on fine-tuning.
