Abstract

SecureCoder is a comprehensive pipeline designed for code generation and evaluation with a strong focus on security awareness. As large language models (LLMs) increasingly assist in software development, ensuring the security of generated code is paramount.

The project features a JetBrains IDE plugin that provides real-time security feedback to developers. Additionally, we introduce a robust benchmark CLI tool capable of evaluating code generation models, available via OpenRouter or running as local agents, against state-of-the-art security benchmarks. Supported benchmarks include CWEval, SecCodePLT, and CyberSecEval.

Project Overview

Security-Aware Pipeline

Our pipeline integrates security evaluation directly into the code generation process, ensuring that generated solutions are not just functional but also resistant to common vulnerabilities.

JetBrains Plugin

The SecureCoder plugin integrates seamlessly into the JetBrains ecosystem. It assists developers by analyzing code in real-time and suggesting secure alternatives or highlighting potential vulnerabilities.

Benchmark CLI Tool

To rigorously test the security capabilities of LLMs, we provide a versatile CLI tool. It streamlines the process of running security benchmarks on various models.

  • Model Support: Compatible with models hosted on OpenRouter as well as local agents.
  • Benchmarks: Includes support for CWEval, SecCodePLT, and CyberSecEval.
